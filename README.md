*Shooting Game* 
==========

遊戲規則
-------------

*_盤   面_  黑色區域為遊戲範圍

*_玩   家_  綠色槍枝

*_按   鍵_  可利用 **左右鍵** 控制槍枝移動，**空白鍵**為攻擊

*_注   意_  請將鍵盤輸入改為**英文**輸入

*_離   開_  離開請按**e**

*_敵   人_  分為三種敵人

  *紅色低級怪物
  *藍色中級怪物
  *白色高階雪怪
